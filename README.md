# IPL-2022-Match-Analysis
An Exploratory Data Analysis (EDA) project on the IPL 2022 dataset to uncover insights into team performance, toss impact, venue trends, player statistics, and match outcomes using Python.

## 📌 Overview
This project analyzes the IPL 2022 season using Python to discover meaningful patterns and trends through data cleaning, exploration, and visualization.

## 📊 Key Analysis
* Dataset Overview
* Team Performance Analysis
* Toss Analysis
* Venue Analysis
* Player Performance Analysis
* Match Outcome Analysis

## 🛠️ Tech Stack
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Repository Structure

